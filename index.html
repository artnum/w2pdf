Convertisseur WhatsApp vers PDF

Programme permettant de convertir le fichier .txt, généré par la fonction "Exporter discussion", en fichier PDF.


Protection des données

Aucune donnée n'est conservée sur le serveur pour la simple et bonne raison que ça ne m'intéresse pas. Si, malgré tout, vous n'avez pas une confiance aveugle, ce qui est raisonnable, vous pouvez me contacter pour une installation sur votre propre serveur. Il est possible de mettre en forme le document résultant selon vos propres critères de présentation.

Bugs

Il est possible que des bugs existent, si c'est le cas, vous pouvez envoyer un extrait du fichier texte en prenant soin de changer les noms des intervenants et de remplacer le texte par des X.
Les smileys ne sont pas supportés pour le moment.
L'incorporation des images n'est pas supporté pour le moment.
